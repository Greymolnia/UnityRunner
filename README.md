## Внесенные изменения

- Добавлены окна меню и паузы, в которых по кнопкам можно либо начать (продолжить) игру, либо завершить.
- Помимо здоровья теперь идет подсчет очков, основанный на пройденном расстоянии + доп очки за поднятые бонусы (5000). Данные очки отображаются и меняются по ходу игры.
- Добавлены два бонуса: первый (зеленый) — пополняет здоровье на 1, если это возможно; второй (синий) — ускоряет передвижение игрока в 1.5 раза на 5 секунд.
- В главном меню сохраняется рекорд по очкам.
- Бонусы спавнятся с вероятностью 0.3 над маленькими и средними препятствиями. Забрать их возможно только прыжком.
- Вероятность для каждого вида бонуса — 50%.

## Новые скрипты

| Скрипт | Описание |
|--------|----------|
| **GamePause** | Меняет сцену и замораживает время |
| **MenuManager** | Отображает лучший результат по очкам |
| **Pause** | Скрипт для сцены паузы, есть функции для перехода в основное меню либо для продолжения игры |
| **PlayerPowerUpHandler** | Отвечает за бонусы, действующие на игроке |
| **PowerUp** | Вызывает добавление очков за бонусы и триггер на сам бонус |
| **PowerUpData** | Базовая структура для всех бонусов |
| **PowerUpSpawner** | Отвечает за корректное появление бонусов над препятствиями |
| **SceneLoader** | Загружает сцену Runner или выходит из игры |
| **ScoreManager** | Считает очки игрока и отрисовывает на экран |
