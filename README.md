Колесник Сергей Александрович

**Описание игры:**
Игрок - куб, который свободно перемещается в ограниченном пространстве. Движение вперед происходит автоматически, возможны одинарные прыжки и повороты влево и вправо.
Существуют три вида препятсвия - Obstacle (маленький, средний и большой куб) с уроном 1, 2, 3 соотвественно. Изначально у игрока 3 здоровья при достижении нуля игра перезапускается.
Генераторы автоматически создают пол и стенки вокруг игрока, пройденные препятсвия удаляются.

**Реализация:**
- **Controller** - скрипт, который дает возможность игроку перемещаться влево и вправо, а также прыгать. Прописано взаимодействие с препятсвиями
- **Follow** - скрипт для следования камеры за игроком.
- **Gen** - создание препятсвий перед игроком, препятсвие, его расположение из трех возможных и удаленность от игрока в заданном диапазоне выбирается рандомно.
- **Health** - отвечает за здоровье игрока и перезапуск игры при смерти игрока. Отображает на экране игрока текущее здоровье.
- **NewGround, NewWalls** - генераторы для стенок и пола.
- **ObstacleData** - наследник для всех препятсвий.
- **Obstacle** - непосредственно препятсвие, с прописанной коллизией.
